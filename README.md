# Credible.io

Credible.io is a real-time credibility verification system designed for education.
It helps students and educators detect, explain, and repair misinformation across
webpages, documents, PDFs, images, and AI-generated content.

## Problem Statement
Combatting Misinformation in Educational Content (TECHFIESTA’26 – ED002)

## What This Project Does
- Verifies factual claims in educational content
- Detects AI hallucinations and fake citations
- Explains *why* content is trustworthy or not (Explainable AI)
- Repairs incorrect information using verified sources
- Helps students learn with accuracy, balance, and confidence

## Core Components
- Browser Extension (content capture & highlighting)
- Backend Verification Engine (fact-checking, bias detection)
- Web Dashboard (credibility scores, explanations)
- Explainable AI Layer (evidence & reasoning)
- Teacher-Governed Trust System

## Tech Stack (Planned)
- Frontend: React / Next.js
- Backend: Python, FastAPI
- AI: LLMs, NLP, LangChain
- Database: Firebase
- Extension: JavaScript (Manifest V3)
